## 文件狀態標記

使用檔名前綴標記文件狀態：

| 標記 | 意義 | 說明 |
|------|------|------|
| `[TD]` | Todo | 待辦，尚未開始 |
| `[DF]` | Draft | 草稿，構想中 |
| `[IP]` | In Progress | 進行中，會持續變動 |
| `[RV]` | Review | 初版完成，待整理 |
| `[FN]` | Final | 完整版，內容穩定 |

**範例：**
- `[DF]新功能規劃.md` - 草稿階段
- `[IP]API開發.md` - 開發中
- `[FN]統一發票概述.md` - 正式文件

**說明：**
- 無標記的文件視為正式文件或參考文件
- `process/` 目錄存放開發過程的暫存文件，完成後可刪除或整理
